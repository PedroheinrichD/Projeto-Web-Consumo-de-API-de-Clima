# 🌤️ Consulta de Clima com JavaScript

Aplicação web desenvolvida com **HTML, CSS e JavaScript** que consome uma API de clima para buscar informações meteorológicas em tempo real de qualquer cidade.

## 🚀 Funcionalidades

- Buscar clima de qualquer cidade
- Exibir **temperatura atual**
- Mostrar **descrição das condições climáticas**
- Exibir **ícone representando o clima**
- Avisar quando a cidade não for encontrada

## 🧠 Conceitos praticados

- Consumo de **API REST**
- Uso da **Fetch API**
- **Async / Await**
- Manipulação de **DOM**
- Estruturação de interface web
